CSCE-313-MP4
============

Multi-threading client-server assignment

####Generate Makefiles
```
cd bin
cmake ../
```

####Running
If you have already generated makefiles
```
cd bin
make mp4
./mp4
```

####Testing
If you have already generated makefiles
```
cd bin
make test
./test_suite
```
NOTE: Google test must be installed or the test suite will not compile.
